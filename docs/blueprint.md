# **App Name**: LinguaLens

## Core Features:

- Text Input: Text area for users to input the text they want to translate.
- Language Selection: Dropdown menus to select source and target languages.
- Translation API Integration: Connect to a translation API (e.g., Google Translate) to perform the translations.
- Translated Text Output: Display the translated text in a dedicated output area.
- Copy to Clipboard: Allow users to copy the translated text to their clipboard.
- Text-to-Speech: Enable text-to-speech functionality for the translated output.
- Translation History (Optional): Store translation history in Firebase.

## Style Guidelines:

- Primary color: Light lavender (#E6E6FA), creating a calming and intellectual atmosphere.
- Background color: Pale gray (#F5F5F5), for a clean and neutral backdrop that doesn't distract from the content.
- Accent color: Soft periwinkle (#CCCCFF) for interactive elements like buttons and links.
- Body and headline font: 'Inter', a sans-serif for a modern and readable interface.
- Use minimalist, monochrome icons for language selection and actions like 'copy' and 'text-to-speech'.
- Clean, responsive layout optimized for both desktop and mobile devices. Use a grid-based system to maintain consistent spacing and alignment.
- Subtle transition animations when displaying the translated text or switching between languages to provide a smooth user experience.
